# ComplaintMS – Complaint Management System

A web-based Complaint Management System that allows users to register, submit, and track complaints while providing administrators with a dashboard to manage and resolve complaints.

## 🌐 Live Demo

https://tanvinaveen04-alt.github.io/complaint-management-system/

## ✨ Features

### User Features
- User registration and login
- Submit complaints
- Select complaint category and priority
- View submitted complaints
- Track complaint status
- Delete complaints

### Admin Features
- Secure admin login
- View all registered complaints
- View complaint details
- Update complaint status
- Mark complaints as resolved

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- LocalStorage
- GitHub Pages
